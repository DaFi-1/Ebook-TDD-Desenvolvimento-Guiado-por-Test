\section*{Sobre o Estudo}
Este repositório é dedicado ao estudo aprofundado do livro apresentado na imagem. A obra traz reflexões e exemplos extremamente plausíveis sobre a metodologia de \textbf{TDD (Test-Driven Development)}.
Inicialmente, havia a percepção de que TDD se resumia ao uso de bibliotecas como \textbf{JUnit}, \textbf{unittest} ou \textbf{pytest}. No entanto, o estudo demonstrou que essas ferramentas são apenas \textit{frameworks de teste}, que surgiram naturalmente a partir de padrões observados por desenvolvedores ao longo do tempo.
Com base no conteúdo do livro, fica claro que TDD é, na verdade, uma \textbf{metodologia estruturada}, com um fluxo bem definido — quase como um ritual. Esse processo pode ser comparado a um rio que, ao encontrar obstáculos, desvia momentaneamente seu curso, mas no final todas as suas partes convergem para um único caminho.

O livro também aborda conceitos relevantes como \textbf{padrões de projeto} e faz menções a \textbf{BDD}. Apesar de alguns capítulos apresentarem explicações mais extensas do que o necessário, o conteúdo geral é extremamente valioso.
De forma geral, trata-se de uma leitura recomendada tanto para iniciantes quanto para desenvolvedores com alguma experiência. O nível não é excessivamente avançado, mas entrega exatamente o que se propõe.

import unittest
from abc import ABC, abstractmethod

class Money(ABC):
    __amount: int

    def equals(self, obj1: Dollar):
        dollar = obj1
        return self.amount == dollar.amount

    def times(self, multiplier: int) -> object:
        self.amount *= multiplier
        return self.amount

    @abstractmethod
    def currency(self) -> str:...

    def set_amount(self, new_amount: int) -> None:
        self.amount = new_amount

    def get_amount(self) -> float:
        return self.amount


class Dollar(Money):
    def __init__(self, amount: int):
        self.amount: int = amount

    def currency(self) -> str:
        return 'USD'

class Franc(Money):
    def __init__(self, amount: int) -> None:
        self.amount: int = amount

    def currency(self) -> str:
        return 'CHF'


class TestDolar(unittest.TestCase):

    def test_multiplication(self) -> None:
        five = Dollar(5)
        five.times(2) 
        self.assertEqual(10, five.amount)
        five.set_amount(5)
        five.times(3) 
        self.assertEqual(15, five.amount)

    def test_franc_multiplication(self) -> None:
        five = Franc(5)
        self.assertEqual(10, five.times(2))
        five.set_amount(5)
        self.assertEqual(15, five.times(3))

    def test_equality(self) -> None:

        self.assertFalse(Dollar(5).equals(Dollar(6)))
        self.assertTrue(Franc(5).equals(Franc(5)))
        self.assertFalse(Franc(5).equals(Franc(6)))
        self.assertFalse(Franc(5).equals(Dollar(6)))

    def test_currency(self) -> None:
